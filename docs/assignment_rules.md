# Assignment rules

この授業では、各課題を GitHub 上で管理します。

## 基本方針

- 配布されたファイルをもとに作業する
- 作業内容は GitHub で管理する
- README や docs も必要に応じて更新する
- 実行できる状態を保つ

## GitHub に含めてよいもの

- Python ソースコード
- README
- docs
- 必要な設定ファイル
- 課題で指定されたサンプルデータ

## GitHub に含めないもの

- 仮想環境 (`.venv`)
- キャッシュファイル (`__pycache__`)
- 一時ファイル
- 実行時に自動生成される不要ファイル

## 初回課題の到達目標

- `setup.sh` を実行できる
- `run_game1.sh` を実行できる
- `game1_shooting/shoot.py` が動作する
- リポジトリの基本構造を理解する
